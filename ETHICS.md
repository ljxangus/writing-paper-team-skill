# Ethics Guidelines — AI-Assisted Academic Paper Writing

## 1. AI Disclosure
- All papers produced with this tool must include appropriate AI assistance disclosure
- Authors remain fully responsible for all content, claims, and data
- AI tools used for writing assistance should be noted in acknowledgments or methodology

## 2. Human Oversight
- All research decisions must be made by human authors
- AI-generated content must be reviewed and validated by domain experts
- Experimental results must be independently verified
- Final paper approval is the sole responsibility of human authors

## 3. Citation Integrity
- No fabricated citations — all references must be verifiable
- AI-suggested citations must be manually verified before inclusion
- Citation context must accurately represent the cited work
- Self-citation patterns must follow community norms

## 4. Data Integrity
- No fabricated or manipulated data
- Experimental results must be reproducible
- Statistical analyses must be correctly performed
- Negative results must be honestly reported

## 5. Anti-Abuse
- This tool must not be used to generate predatory or fraudulent publications
- Paper-mill behavior (mass producing low-quality papers) is prohibited
- Plagiarism detection should be run on all output
- The tool should not be used to circumvent peer review standards

## 6. Responsible Use
- Users must have genuine research intent
- Output quality is the user's responsibility
- The tool assists but does not replace research effort
- Users should contribute intellectual input at each stage
